En el primer ejercicio la clase Persona tiene dos atributos nombre y edad con private, lo que significa que nadie puede tocarlos directamente desde Main. Para leerlos se usan los get y para modificarlos los set.
En el Main primero se crea el objeto con "Alex" y 21, se muestra su información, luego se cambia el nombre a "Paul" y la edad a 25 usando los set, y se vuelve a mostrar para ver el cambio.

En el segundo ejercicio igual que el anterior pero aplicado a un producto de tienda. Los atributos nombre y precio son private, nadie los puede modificar directamente.
En el Main se crea el producto "Arroz" con precio 1.50, se muestra, luego se actualiza el precio a 2.00 con setPrecio() y se vuelve a mostrar.
